Task -1 Online Reservation System 

A simple Java console-based Online Reservation System that simulates basic railway ticket booking operations.
This project demonstrates authentication, ticket reservation, cancellation, and file-based data storage using Core Java.

1. User Login & Registration

2. Ticket Reservation

Enter passenger details, train number, class type, journey date, source, and destination.

3. View Reservations

Users can view all their previously booked tickets.

4. Ticket Cancellation

Cancel using PNR.

Displays full ticket details before confirmation.

Only the owner of the PNR can cancel the ticket.

Simulates a small database using:
